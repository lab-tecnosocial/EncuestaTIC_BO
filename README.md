# EncuestaTIC_BO
Datos en formato abierto de la Encuesta TIC de 2017 en Bolivia. 

Los datos se encuentran en la carpeta `csv`, con el siguiente contenido:

```
├── eti
│   └── datos_etiquetas_todo.csv
├── sem
│   ├── base.csv
│   └── dic.csv
└── val
    ├── datos_valores.csv
    ├── etiquetas_valores.csv
    └── etiquetas_variables.csv
```
